# Plano de Ação - Trilha-SDR
Plano de ação para sanar gaps de vagas no time de SDR. A ideia é criar uma trilha de SDR para que jovens talentos tenha a primeira oportunidade no mercado de trabalho ou pessoas que queiram fazer uma transição para a área comercial. A ideia é criar um programa de desenvolvimento onde no final do projeto a colab esteja pronta para vagas no comercial
